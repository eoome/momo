# ModernWMS 系统启动说明

## 已修复的问题
1. 修改了 `RolemenuService.cs`，移除了对不存在数据库字段的查询
2. 避免了登录后的菜单权限查询错误

## 启动方式

### 方式一：直接运行编译好的程序

**启动后端：**
```powershell
cd c:\Users\Administrator\Desktop\WMS\WMS\backend\ModernWMS\bin\Debug\net7.0
.\ModernWMS.exe
```
后端会在 http://localhost:5555 启动

**启动前端：**
```powershell
cd c:\Users\Administrator\Desktop\WMS\WMS\frontend
npm run dev
```
前端会在 http://localhost:5173 启动

### 方式二：使用 dotnet run 启动后端
```powershell
cd c:\Users\Administrator\Desktop\WMS\WMS\backend\ModernWMS
dotnet run --urls "http://localhost:5555"
```

## 登录信息

## 数据库
- 数据库：SQLite
- 数据库文件位置：`backend\ModernWMS\bin\Debug\net7.0\wms.db`

## 注意事项
- 请同时启动后端和前端
- 确保端口 5555 和 5173 没有被占用
